# Azure Enterprise SOC & Cloud Security Lab

## Overview

This project demonstrates the deployment and operation of a cloud-based Security Operations Center (SOC) using Microsoft Azure.

The environment includes Microsoft Sentinel, Microsoft Defender for Cloud, Microsoft Entra ID, Azure Monitor, Log Analytics, Azure Policy, and enterprise networking to simulate a production cloud environment.

---

## Objectives

- Build a secure Azure environment
- Monitor Windows and Linux workloads
- Detect malicious activity using Microsoft Sentinel
- Hunt threats using KQL
- Automate incident response
- Implement Zero Trust principles
- Secure cloud workloads using Microsoft Defender for Cloud
